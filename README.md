# KJ_LookinServer
手动配置Lookin Version1.0.7

Lookin图层分析工具
## 1、[Lookin官网](https://lookin.work)下载安装Mac Lookin
## 2、引用LookinServer源码文件夹
## 3、在Build Setting/Apple Clang - Preprocessing / Preprocessor Macros Debug增加编译参数SHOULD_COMPILE_LOOKIN_SERVE
